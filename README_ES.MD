# Ejemplo de Github Pages
Un prototipo funcional sencillo de un sitio web.

`[SimPIL son una serie de ejemplos ultra-sencillos de cómo realizar diferentes operaciones con código. Sin complicaciones.]`

## Configuración
En **Settings / Pages** (a la izquierda), selecciona como fuente *"Deploy from a branch"* y elige la rama (por ejemplo, `master`) y la ruta (`"/"` o la que corresponda).  
Después podrás acceder al sitio en [Prototipo Funcional de Sitio Web](https://davidgbonacho.github.io/Sample-Github-Pages/).

## Archivos
Solo son los archivos del prototipo funcional:  
- `index.html`  
- `listadovideos.html`  
- `video.html`
